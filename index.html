Publication

I published a research paper named Driver Drowsiness Detection Using Artificial Intelligence. During which I learnt a lot about the usage of OpenCV. The system so designed is a non-intrusive real-time monitoring device. The priority is on improving the safety of the driver without disturbing him. Eyeblink of the driver is detected in our Paper. The programming for that is achieved in OpenCV using the Haarcascade library for the detection of facial features
